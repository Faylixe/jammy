Installation


Jammy requires Java 8 and Eclipse Luna

Linux users will have some issues with Eclipse integrated webbrowser if they use GTK, see here for more details

Using update site


First add Jammy repository : in Eclipse, go to Help > Install new software..., click on Add button and enter the following url :


http://faylixe.fr/jammy/update


Now you can use this repository to install Jammy and all addons availables.

Manually


You can also directly download plugin from here, and move it to Eclipse plugin directory.
