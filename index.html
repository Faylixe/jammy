Jammy is a set of Eclipse plugin for managing Google Code Jam development.


Custom perspective

Including dedicated view such as problem description, submission monitoring, or contest explorer.


Automatic submission

Downloads input, runs it, and uploads packaged output with source code without any effort.


Sample dataset extraction

Sample dataset is created from problem description so you can test your algorithm quickly.

Basic plugin installation provides support for Java language only, but any programming language could be supported, using the Addon API.
