Where should i start ?


Once Jammy is installed, you just have to switch to the associated perspective using menu Window > Open perspective > Other. You can then authenticate to Google service using Jammy > Login. Finally you can choose a contest by either navigating through archive index, or by giving the contest identifier.


How can i write my solution ?


When a contest is selected, the Context Explorer view shows each problem instance available. Double click on a problem or using contextual menu Open problem solver.


Programming language selection


Jammy plugin is cross language, it could work on any programming language as long as a language extension plugin is installed. For selecting the language you wish to work with, go to Preference > Jammy and select it through the proposed drop down list.


Supported language(s)

Here is a list of known language supported by Jammy :


Java

Java is the default language supported by Jammy. Indead Jammy is coded in Java, for a Java first based IDE.


You need another language ? Try to integrate it using the Language extension API and we will add it to this list !


My algorithm is ready, how to submit it ?


You can use the contextual menu on any problem you are working on to submit your solution. Right click on your problem input you want in the Context explorer view, and select Submit.
